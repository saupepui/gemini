Transforma este proyecto en una plantilla base vacía (Boilerplate) de Astro y Tailwind. Instrucciones: 1) Elimina mediante comandos de terminal los archivos y carpetas específicas del proyecto anterior: 'src/data/coches.json', 'src/components/Simulador.astro' y el directorio entero 'src/pages/coches/'. 2) Edita 'src/pages/index.astro' para que sea una página limpia y genérica. Debe contener un <main> centrado con diseño moderno de Tailwind, un <h1> que diga 'Plantilla Base Astro + Tailwind' y un párrafo que diga 'Proyecto inicializado y listo para nuevas instrucciones del agente autónomo'. Conserva el script de Tailwind 'is:inline' en el <head>. 3) No toques los archivos de configuración (package.json, astro.config.mjs, etc.). Haz git add, git commit --no-gpg-sign con el mensaje 'Refactor: Limpieza total para crear Plantilla Maestra' y git push origin main.
