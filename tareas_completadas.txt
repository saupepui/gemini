Crea un proyecto web inicial usando Astro. Ejecuta en bash este comando exacto: 'npx --yes create-astro@latest web_coches --template minimal --no-install --no-git --yes'. Una vez termine, entra en 'web_coches', ejecuta 'npm install' y luego 'npx --yes astro add tailwind --yes'. Mueve todos los archivos (incluyendo ocultos) desde 'web_coches' a la raíz del repositorio y borra la carpeta vacía. Prueba que compila con 'npm run build' y haz el commit inicial y push.
Crea el contrato de datos. Genera un archivo en 'src/data/coches.json'. Debe contener un array JSON con dos objetos reales (ej. Tesla Model 3 y Dacia Spring). Cada objeto debe tener exactamente estas claves: 'id' (un slug como 'tesla-model-3'), 'marca', 'modelo', 'bateria_util_kwh', 'consumo_medio_kwh_100km' y 'potencia_carga_dc_max_kw'. Haz el commit y push.
Crea la lógica matemática. Crea un archivo 'src/utils/calculadoras.js' que exporte 3 funciones puras: 'calcularTiempoDomestica(bateria, potenciaCargador)', 'calcularCoste100km(consumo, precioKwh)' y 'calcularTiempoRapida(bateria, potenciaDcMax)'. Crea un script temporal para probar que las matemáticas devuelven valores lógicos por consola, bórralo, y haz commit y push.
Crea la generación de páginas dinámicas (SEO). En Astro, crea el archivo 'src/pages/coches/[id].astro'. Usa 'getStaticPaths' para importar el 'src/data/coches.json' y generar una ruta por cada coche. El HTML debe mostrar un h1 con la Marca y el Modelo, y unos estilos básicos usando clases de Tailwind. Ejecuta 'npm run build' para asegurar que se generan las carpetas estáticas. Haz commit y push.
Crea el frontend interactivo. Crea un componente 'src/components/Simulador.astro'. Este componente recibirá el objeto del coche como Prop. Diseña una tarjeta limpia con Tailwind que tenga etiquetas <input> HTML para que el usuario escriba su 'Precio del kWh' y su 'Potencia de carga'. Añade una etiqueta <script> con Vanilla JavaScript que escuche los cambios en esos inputs, importe las funciones de 'calculadoras.js' y actualice el DOM con los resultados en tiempo real. Haz commit y push.
Conecta todo. Importa e inserta el componente 'Simulador.astro' dentro de la plantilla 'src/pages/coches/[id].astro' pasándole los datos del coche actual. Luego, edita 'src/pages/index.astro' para que lea el JSON y muestre un listado (grid) de todos los coches disponibles con enlaces a sus respectivas páginas dinámicas. Ejecuta 'npm run build' para verificar todo el sitio completo, haz commit y push.
